## Getting Started

Als erstes navigiert man in den Ordner 'event-shop-app'.

Der nächste Step ist die App lokal zu starten, dafür verwendet man den Befehl:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
